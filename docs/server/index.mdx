# Python 服务器开发

这一部分主要学习如何使用 Python 开发服务器程序。

服务器开发不是只写几个接口，它还会涉及请求处理、路由设计、数据库访问、认证授权、配置管理、日志记录、异常处理、部署运行等内容。掌握这些知识后，就可以用 Python 编写后端接口、管理后台、自动化服务、数据服务和内部工具。

## 你会学到什么

- Web 服务的基本工作流程
- HTTP 请求和响应的核心概念
- API 接口的设计方式
- Python 常见 Web 框架的使用
- 数据库连接和数据读写
- 登录、权限和安全相关知识
- 日志、配置和错误处理
- 服务部署和运行维护

## 学习前准备

建议先掌握 Python 基础语法，再学习服务器开发。至少需要熟悉：

- 变量、函数、模块和包
- 面向对象基础
- 虚拟环境和依赖管理
- 文件操作和异常处理
- 命令行的基本使用

如果这些内容还不熟，可以先回到 Python 基础部分学习。

## 学习路线

推荐按照下面的顺序学习：

1. 先理解 HTTP 和服务器的基本概念
2. 再学习如何用 Python 启动一个 Web 服务
3. 接着学习路由、请求参数、响应数据和接口设计
4. 然后接入数据库，完成真实的数据增删改查
5. 最后学习认证、日志、配置、部署等工程化内容

:::tip 提示
服务器开发关注的是“程序如何对外提供服务”。写代码只是其中一部分，更重要的是理解请求从客户端到服务器，再从服务器返回响应的完整过程。
:::
