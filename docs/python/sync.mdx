## 故事的背景
不同于Java或C++等语言，Python是一门单线程语言，这意味着它不能利用Cpu的多个核心同时执行多个任务，同一时间只能执行一个任务，无法同时并发执行多个任务。
![250](https://img.dingshaohua.com/file/1778065676184_just_me_do.jpg)


这意味着，如果一个任务需要很长时间才能完成，那么其他任务就必须等待，直到这个任务完成。同步异步指的是程序运行的方式：正常情况下，程序都是从上到下按顺序执行的。

在同步模式下，这是一种‘阻塞式’的顺序：每一行代码都必须‘等’上一行代码彻底执行完、拿到结果后，才能开启下一行。就像接力赛，棒子没传到手，你就只能原地等。
